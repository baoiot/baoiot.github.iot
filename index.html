Cài đặt Bootloader

    OpenCore Bootloader EFI for all Intel CPU generation and AMD CPU with UEFI/Legacy boot mode on Desktop or Laptop

  • Để cài OpenCore Bootloder lên USB hoặc ổ cứng, bạn chỉ việc giải nén EFI của cộng đồng đã làm sẳn ở trên vào EFI của USB hoặc ổ cứng là xong

Tinh chỉnh Config

  • Nếu Desktop dùng Intel CPU thì bạn hãy tạm thời bỏ qua bước này (trừ CPU Pentium và Celeron thì bạn cần Fake CPU), còn AMD CPU thì bạn hãy xoá config.plist gốc ở /EFI/OC/ rồi rename config2.plist thành config.plist
  • Với Laptop thì bạn hãy vào /EFI/OC/config_laptop/, tìm đến config chứa tên iGPU của máy bạn, rename nó thành config.plist sau đó thay thế config gốc ở /EFI/OC/
